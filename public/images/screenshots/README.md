# Captures d'écran PronoFootball.Club

Placez vos captures d'écran de l'application dans ce dossier.

## Fichiers recommandés :

- `dashboard.png` - Capture du tableau de bord principal
- `betting.png` - Capture de la page de pronostics
- `stats.png` - Capture de la page de statistiques
- `competitions.png` - Capture de la page des compétitions

## Format recommandé :

- Format : PNG ou JPG
- Taille : 1200x800px ou similaire (ratio 3:2)
- Qualité : Haute résolution pour un rendu net

Une fois les fichiers ajoutés, décommentez la section "Découvrez l'interface" dans `pages/index.tsx`.
